Résumé Exécutif
Vue d'ensemble de la plateforme

LogiSaaS est une plateforme SaaS cloud-native de gestion du fret et de l'entrepôt, conçue pour concurrencer directement CargoWise et Magaya sur l'expérience utilisateur. L'avantage différenciateur n'est pas la liste de fonctionnalités, mais la philosophie UX : les utilisateurs voient uniquement ce qui nécessite une décision, pas un tableau de bord de tout.

Architecture event-driven avec Kafka comme épine dorsale, multi-tenant PostgreSQL avec Row-Level Security, et une UX centrée sur 3 patterns différenciateurs : Exception Inbox Zero, Omni-search Cmd+K, et le Kanban drag-and-drop des shipments.

Risques & Mitigations
Points de vigilance identifiés dans le blueprint technique
RisqueImpactMitigation
EDI malformé carrier HAUTE Validation par message (pas par batch) · Dead-letter topic · Auto-ticket ops · Replay tooling
Write contention WMS scanning HAUTE CQRS append-only scan_events · Projector async · Optimistic concurrency · Read replicas
Multi-currency rounding drift MOYENNE NUMERIC(14,6) haute précision partout · Banker's rounding uniquement à l'affichage
Webhook/event replay doubles HAUTE Idempotency keys (processed_events table) · Tous consumers safely re-runnable
Timezone ambiguity milestones MOYENNE Normalisation UTC à la frontière d'ingestion · Raw string conservé pour audit
Tenant data leakage (cache) HAUTE tenant_id dans toutes les cache keys · Jamais de cache au-dessus de la résolution tenant
